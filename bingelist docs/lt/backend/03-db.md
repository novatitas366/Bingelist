# src/db.js — Duomenų bazė

**Šaltinio failas:** `src/db.js`  
**Priklauso nuo:** niekas (naudoja Node integruotus modulius + `better-sqlite3`)  
**Naudoja:** [[07-routes-auth]], [[09-routes-watchlist]], [[10-routes-episodes]], [[02-server]]

---

## Ką šis failas daro

Atidaro (arba sukuria) SQLite duomenų bazės failą ir pirmojo paleidimo metu sukuria tris lenteles. Eksportuoja:
- `db` — duomenų bazės ryšys naudojamas visuose maršrutų failuose
- `isUniqueViolation(e)` — pagalbinė funkcija duplikatų klaidoms aptikti

---

## Pagrindinės sąvokos

- **SQLite** — failų pagrindu veikianti reliacinė duomenų bazė. Visa duomenų bazė gyvena `watchly.db` faile. Atskiro duomenų bazės serverio nereikia.
- **better-sqlite3** — sinchroninis SQLite tvarkyklė. Skirtingai nuo daugumos Node duomenų bazių bibliotekų, užklausos negrąžina Promise — jos tiesiogiai grąžina rezultatus.
- **Paruošti teiginiai** — `db.prepare(sql)` kompiliuoja SQL vieną kartą. Gautas teiginys tada kviečiamas su `.run()`, `.get()` arba `.all()`. Parametrai naudoja `?` vietos žymeklus SQL įterpimo prevencijai.
- **WAL režimas** — Write-Ahead Logging. Keli skaitytuvai gali dirbti vienu metu su vienu rašytoju neblokuojant vienas kito.
- **Išoriniai raktai** — pagal nutylėjimą išjungti SQLite; pragma juos įjungia kad `ON DELETE CASCADE` veiktų.

## Lentelės

### `users` (vartotojai)
| Stulpelis | Tipas | Pastabos |
|---|---|---|
| `id` | INTEGER PK | automatiškai didėja |
| `username` | TEXT UNIQUE | turi būti unikalus, 3–32 simboliai |
| `password_hash` | TEXT | bcrypt hešas — niekada ne paprastas slaptažodis |
| `created_at` | TEXT | `datetime('now')` numatytoji reikšmė |

### `watchlist`
| Stulpelis | Tipas | Pastabos |
|---|---|---|
| `id` | INTEGER PK | |
| `user_id` | INTEGER FK → users | `ON DELETE CASCADE` |
| `show_id` | INTEGER | TVMaze serialo ID |
| `show_name` | TEXT | talpykloje saugoma kad nereikėtų visada iš naujo gauti |
| `show_image` | TEXT | gali būti null |
| `status` | TEXT | viena iš 4 leidžiamų reikšmių (CHECK apribojimas) |
| `note` | TEXT | gali būti null, vartotojo asmeninė pastaba |
| `added_at` | TEXT | `datetime('now')` numatytoji reikšmė |
| UNIQUE | `(user_id, show_id)` | vienas įrašas vienam serialui vienam vartotojui |

### `watched_episodes` (pažiūrėti epizodai)
| Stulpelis | Tipas | Pastabos |
|---|---|---|
| `id` | INTEGER PK | |
| `user_id` | INTEGER FK → users | `ON DELETE CASCADE` |
| `show_id` | INTEGER | TVMaze serialo ID (greitoms užklausoms pagal serialą) |
| `episode_id` | INTEGER | TVMaze epizodo ID |
| UNIQUE | `(user_id, episode_id)` | negalima pažymėti to paties epizodo du kartus |
